HTML Forms

El elemento <form> es un contenedor para diferentes tipos de elementos de entrada, como: campos de texto, botones, checkboxes, botones de opción, etc.

El elemento HTML <input> es el elemento de formulario más utilizado.
Un elemento <input> se puede mostrar de muchas maneras, según el atributo de type.
Para marcar un campo como necesario, agrega el atributo required en la etiqueta de input.

El radio button o botones de opción permiten al usuario seleccionar UNA sola opción de una lista y se define como <input type="radio">

Los checkboxes o casillas de verificación permiten al usuario seleccionar CERO o VARIAS opciones de un número limitado de opciones.
<input type="checkbox"> define una casilla de verificación.

El elemento <select> define una lista desplegable, es decir, un dropdown.
Para definir una opción preseleccionada, agregue el atributo selected a la opción.

El elemento <textarea> define un campo de entrada de varias líneas de texto.
El atributo de rows especifica el número visible de líneas en un área de texto.
El atributo cols especifica el ancho visible de un área de texto.
También puedes definir el tamaño del área de texto usando CSS con las propiedades width y height

El <input type="date"> se utiliza para campos de entrada que deben contener una fecha.

Para agregar un campo de correo electrónico se utiliza <input type="email">

Existe un botón específico para enviar los datos en una forma.
<input type="submit"> define un botón para enviar los datos a un controlador de formulario.
Utiliza el input de tipo <input type="reset"> para restablecer las respuestas del formulario.

Para más información, visita: W3 Schools: HTML Forms
